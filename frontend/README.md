# ChatDock

ChatDock is a Next.js website for a hackathon chatbot maker. It lets teams design a website assistant, configure model fallback through TrueFoundry AI Gateway, scope MCP tools, add guardrails, set budgets, and preview the generated embed/config files.

The website UI direction was adapted from an older project layout, then reworked with new ChatDock content, chatbot maker flows, TrueFoundry hackathon positioning, and compact product screens.

## Run

```bash
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) when running the local dev server on port 3001.

## Main Routes

- `/` - public product page
- `/builder` - chatbot configuration builder
- `/templates` - reusable chatbot templates
- `/architecture` - TrueFoundry architecture mapping
- `/demo` - judge-facing demo plan
- `/about` - hackathon brief
